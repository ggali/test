The simple system

Developer creates the repository on Github and set permissions

Users create their own pull request.That is a copy of the master where they can freely iterate. 

Users modifiy the pull request by the simple bootstrap editor. Each user according to its own role and skill.
Copywriters modifies texts, designers act on colors or the UI, developers add animations or whatever
