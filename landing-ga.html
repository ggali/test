Github flow

Developer
Set up Github

Developer creates the repository on Github and set permissions.
Github is where developers live. Your pages are here ready to get to any following step, published on GH pages, pushed to AWS or integrated anywhere by tons of ready made Github connectors or by custom code.

Users
Make their own copy

Users create their own pull request.
A pull request is a copy of the master where users can freely iterate. 

Users
Make changes

Users modify the pull request by the simple web editor. Each one according to its own role and skill.
Copywriter modifies texts, designer act on colors or the UI, developer add animations or whatever required. 

Developer or Project owner
Accept or deny changes

Developer and the project owner accept or deny the pull requests.
